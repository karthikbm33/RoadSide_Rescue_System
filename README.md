## рҹҡЁ About The Project

<table>
<tr>
<td>

**RoadSide Rescue System** is an emergency response web application designed to help people during road accidents and vehicle breakdowns. It provides a fast, reliable way to connect victims with nearby rescue services, mechanics, and emergency contacts.

> рҹҺҜ **Mission** вҖ” Every second counts in an emergency. This system ensures help reaches people as fast as possible.

### рҹ’Ў Problem It Solves:
- рҹҡ— Vehicle breakdown in remote areas
- рҹҸҘ Accident emergency response
- рҹ”§ Quick mechanic / tow truck finder
- рҹ“һ Emergency contact & SOS alerts
- рҹ—әпёҸ Real-time location sharing
